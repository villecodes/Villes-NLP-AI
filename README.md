## Get started
* Run ``` npm install ```.
* Run ``` node index.js ```.

## Change lyrics or text
* Create a new .text file with your lyrics / text.
* __[In the index.js file]__ change the ``` document: readFile('./shook-ones-lyrics.txt') ``` to your filename.
